Web page with static content
Static content is any content that can be delivered to the end user without having to be generated, modified, or processed. The server delivers the same file to each user, making static content one of the simplest and most efficient content types for transmission. Since no modification/processing is required, there is no need to use MVC controllers for handling the HTTP request.

This kind of content could be stored in any memory device supported by Elysian (ROM, RAM, Handler or Ext(ernal) device). As can be seen for the URL prefix (/fs_rom/), this particular Web Page is stored in the ROM memory device using the "makefsdata.py" tool.
