HTTP request exposure
There are times when full access to the HTTP request header or/and body is required. This could be true if for example there is a need to send different version of the web page when the HTTP client is a mobile phone, or if the HTTP body is a JSON request. The HTTP request headers/body can be exposed as parameters with certain names. The controller of the example below is retrieving the HTTP headers of the request, and sends them back to the client when the "submit" button is pressed.
