HTTP request exposure
Elysian provides full access to the HTTP request contents.

The web application is able to perform different actions based on the contents of the URL, HTTP headers and the HTTP body. It is for example possible to provide different versions of a web application depending on whether the HTTP request was originated by a mobile phone or a desktop computer.
