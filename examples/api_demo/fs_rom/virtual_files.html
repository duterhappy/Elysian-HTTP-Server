Virtual memory device
Except from the RAM, ROM and EXT(ernal) memory devices, Elysian Web Server also supports virtual files.

A virtual file is not an actual file and it is not stored in any memory device. It is implemented using a function handler that generates the file content dynamically upon request.

The virtual memory device can be used to generate small or large files on-the-fly without taking up any space in the executable or any other storagte device.
