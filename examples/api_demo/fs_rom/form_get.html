HTTP form submission, using HTTP GET method
Forms are used to collect data input from a user.

To send the data submitted through a form, an HTTP client (Web Browser) can use the GET or POST HTTP method.

The GET method sends the encoded user information appended to the page request. The page and the encoded information are separated by the "?" character.

The POST method transfers information via HTTP headers. In most browsers the submitted form data are typically not visible to the user.

This particular example uses the HTTP GET method. The submitted data are send back to the browser for display.
