HTTP basic access authentication
Basic access authentication is a method for a HTTP client to provide a user name and password when making a request.

When the Web server receives an anonymous request for a protected resource it will force the use of basic authentication by rejecting the request.

Most web browsers will display a login dialog, allowing the user to enter a username and password.
