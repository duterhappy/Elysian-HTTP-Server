Welcome
Elysian is a free, open source and extremely lightweight Web Server for embedded devices and microcontrollers.

The focus of the implementation is to reduce resource usage while still providing an almost full-scale HTTP Server, making it suitable for use in memory-constrained embedded devices where memory and CPU power are at a premium.

The Web Server allows to serve richfull Web Pages with images and data, enabling embedded monitoring and control functionality into the deployed TCP/IP enabled appliances.

Much emphasis is given on ease of use, easy integration and robustness.
