Web page with dynamic content
Dynamic content is the content that changes between subsequent requests based on user access time, user preferences and other factors in order to deliver an engaging online experience. To deliver dynamic content, Elysian Web Server adopts the MVC (Model-View-Controller) design pattern. MVC offers architectural benefits and helps to write better organized, and therefore more maintainable code.

The Model encapsulates the data and defines the logic and computation that manipulate and process that data.

The View is what's presented to the users and how users interact with the app. The view is made with templates of HTML, CSS or JavaScript content.

The Controller is the decision maker and the glue between the model and view. Typically it implements the application logic.
