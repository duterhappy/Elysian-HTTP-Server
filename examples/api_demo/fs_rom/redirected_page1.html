HTTP redirection
To increase flexibility, Elysian offers the ability to alter the HTTP status code and HTTP headers of a particular HTTP response, to implement a particular HTTP feature. In this example, the application layer replaced the default "HTTP/1.1 200 OK" with a custom HTTP status code ("HTTP/1.1 302 Moved") and also put an additional HTTP response header ("Location") to implement the HTTP redirection feature.

HTTP redirection allows servers to redirect a client request to a different location. Even though the "HTTP redirection" menu item was referring to 'redirected_page0.html', when pressed, user was actually redirected to 'redirected_page1.html'.
